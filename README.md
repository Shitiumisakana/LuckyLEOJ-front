# LuckyLEOJ 前端管理系统 (LuckyLEOJ-front)

本项目是 LuckyLEOJ 判题系统的配套前端工程，基于 **React 18**、**Ant Design Pro** 和 **Umi 4** 构建。

* **前端工程**: [LuckyLEOJ-front](https://www.google.com/search?q=https://github.com/Shitiumisakana/LuckyLEOJ-front)
* **后端核心**: [LuckyLEOJ](https://github.com/Shitiumisakana/LuckyLEOJ)
* **代码沙箱**: [LuckyLEOJ-sandbox](https://www.google.com/search?q=https://github.com/Shitiumisakana/LuckyLEOJ-sandbox) (基于 Docker 的安全判题环境)

## ✨ 核心特性
* **编辑器集成**：集成 **Monaco Editor**，支持多语言代码高亮与编辑。
* **高级组件**：基于 **ProComponents** 封装题目管理看板、判题记录表格。
* **状态管理**：利用 Umi 内置插件实现全局用户登录态（分布式 Session）的维护。
* **内容支持**：支持 **Markdown** 格式的题目描述展示。

## 🛠️ 技术栈
* **框架**: React 18, Umi 4
* **UI**: Ant Design, ProComponents
* **请求**: Umi Request (Axios 封装)
* **工具**: TypeScript, Monaco Editor

## 🚀 快速启动
1. 安装依赖: `npm install`
2. 启动项目: `npm run start:dev`
3. 打包项目: `npm run build`
